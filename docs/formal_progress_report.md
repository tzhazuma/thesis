# 课题阶段性进展报告

## 一、课题进展情况（含已取得的阶段性成果）

本课题围绕“基于高斯和分解与随机批量采样的长程静电算法重构”展开，主要面向超大规模全细胞分子动力学模拟中的关键瓶颈问题，即传统基于 Ewald 分离和 FFT 的长程静电求解方法在大规模并行环境下容易受到跨节点通信与同步开销的显著制约。结合近年来高斯和分解、张量表示以及随机批量方法在高维科学计算中的发展趋势，本课题试图探索一种兼顾数值平滑性、统计稳定性与算法扩展潜力的新型长程相互作用处理框架，并以 RBSOG 方法作为当前阶段的核心研究对象。

从总体进展来看，课题已顺利完成前期方法论梳理、实验路径设计、原型系统搭建、初步性能优化以及阶段性结果组织等工作，研究已经从概念设想和方案论证阶段，进入到了“原型可运行、结果可分析、报告可生成”的实质性推进阶段。当前工作的突出特点在于，不仅完成了方法实现本身，而且已同步建立起了一条具有较高复现性的实验评测与论文写作支撑链条，为后续进一步深化研究和形成学术成果奠定了坚实基础。

首先，在研究对象与实验路线方面，已完成适用于个人计算机环境的最小验证体系构建。考虑到真实全原子生物膜体系以及全细胞模型在开发早期会引入复杂的参数、力场和软件耦合问题，当前阶段采用了膜启发的代理体系作为验证对象。该体系在规模和建模复杂度上更适合原型研究，但仍保留了课题最关心的核心特征，包括长程静电相互作用、显式溶剂、NPT 风格统计量以及结构稳定性相关观测指标。这种设计思路保证了研究能够聚焦于算法本身，而不被过早卷入过多的工程细节之中，同时也为未来逐步迁移到更真实的 DPPC 膜体系和更复杂的生物膜系统预留了明确路径。

其次，在算法框架构建方面，已经完成了统一研究原型的主体搭建。当前原型系统实现了三类长程求解器：一是用于小规模参考校验的直接求和模块，二是作为传统基线方法的 PPPM 模块，三是本课题重点研究的 RBSOG 模块。RBSOG 的当前实现采用“短程确定性修正 + 长程高斯和随机批量采样”的基本思路，在统一的模拟框架下与 PPPM 进行对照评测。与此同时，项目还构建了完整的命令行入口、模拟配置接口、结果记录模块、统计分析模块和可视化导出模块，使整个研究过程从单次试验脚本演进为一个具备持续迭代能力的科研原型平台。这一成果标志着课题已经不再停留于概念推演，而是形成了可以反复运行、持续扩展和支撑论文写作的技术基础。

再次，在性能优化与工程实现方面，课题已取得一系列阶段性改进。前期实验表明，RBSOG 原型中的主要开销来自短程相互作用的重复近邻搜索以及长程采样在 Python 层的逐项累积计算。针对这一问题，当前已完成近邻表短程求解器的引入，并实现了长程采样部分的向量化改造和可选 numba JIT 加速路径。通过这些优化，RBSOG 的运行效率相较最初原型已有明显提升，说明该方法在进一步工程化后具有较好的性能挖掘空间。虽然在当前小规模、纯 Python 原型条件下，RBSOG 尚未在绝对步耗时上全面超过 PPPM，但现有结果已经明确显示其在压力方差控制与结构稳定性指标方面具有积极表现，这为后续深入优化提供了充分依据。

在实验评测体系建设方面，课题目前已经形成了较为完整的自动化流程。该流程能够支持单次模拟、PPPM 与 RBSOG 多随机种子基准对比、RBSOG 批量大小扫描、JIT 消融实验以及结构稳定性分析。围绕批量大小 P 的选择问题，当前系统已进一步加入 Pareto 前沿识别与加权推荐机制，可依据运行时间和压力方差两个目标自动给出推荐参数。这一设计有效提升了方法学研究的客观性，使参数选择不再依赖经验判断，而可以通过明确的评价指标和优化准则完成。此外，现有实验输出已经覆盖平均每步时间、压力方差、温度统计、能量漂移、面积/脂质波动、膜厚 proxy 漂移等关键量，为后续精细化分析与论文撰写提供了足够的数据支撑。

在成果组织和学术表达方面，课题也已取得较明显进展。基于当前原型与实验结果，已经建立了一套可自动生成图表、表格和宏定义的 LaTeX 报告流程，并成功编译形成会议风格的 PDF 文稿。报告内容已覆盖研究背景、方法设计、实验设置、基准结果、参数推荐、消融实验、结构稳定性分析以及可复现性说明等核心章节。近期又进一步补充了运行级统计、效应量分析和敏感性图，使文稿质量更加接近正式学术论文的组织方式。这表明课题目前已经从“原型开发”进一步迈向“证据组织与成果表达同步建设”的阶段，为后续论文投稿和阶段性汇报提供了成熟基础。

综合而言，本课题当前已完成从研究问题凝练、技术路线设计、原型系统搭建、性能初步优化、实验流程建设到学术报告生成的一整套前期工作。阶段性成果可以概括为以下三个方面：其一，成功构建了面向长程静电求解问题的 RBSOG 研究原型；其二，形成了兼顾运行效率、统计方差和结构稳定性的综合评测体系；其三，建立了实验与论文写作联动的自动化支撑流程。上述成果为课题下一阶段向更高真实性体系、更严格统计分析和更深层算法优化推进提供了可靠基础。

## 二、存在的问题及解决思路

尽管课题目前已取得较为明确的阶段性成果，但从研究目标和学术产出的要求来看，仍存在若干需要进一步解决的问题。这些问题既反映了方法原型阶段的客观局限，也明确指出了课题下一步深化研究的主要方向。总体而言，当前问题主要集中在模型真实性不足、绝对性能仍待增强、统计证据不够充分、复杂度论证尚不完整以及精度分析有待系统化等几个方面。

首先，当前研究所采用的体系仍属于膜启发代理模型，尚未完全进入真实全原子膜体系或更复杂的全细胞模型验证阶段。这种代理模型对于前期方法开发具有显著优势，能够帮助研究者集中精力解决算法实现、参数控制和指标设计等关键问题，但与此同时，其物理真实性和生物化学细节表达能力仍然有限。因此，现阶段所得结论更多体现为方法学与算法工程上的可行性证据，还不足以直接外推到真实复杂生物体系中。针对这一问题，后续应坚持“保持评测框架稳定、逐步提升体系真实性”的思路，即在不改变当前 benchmark 与分析管线的前提下，逐步引入更真实的膜初态、更规范的参数设置和更复杂的体系构型，最终将现有方法迁移到更接近真实应用背景的环境中验证。

其次，RBSOG 当前在小规模纯 Python 原型中的绝对运行速度尚未全面优于 PPPM。需要指出的是，这一现象具有较强的阶段性特征，并不构成对研究路线的否定。PPPM 在现有数值库支持下已经非常成熟，而 RBSOG 目前仍处于研究原型阶段，其长程随机采样内核、数据布局与加速路径尚未得到彻底工程化，因此更适合用于验证算法潜力，而非直接代表方法的最终性能上限。当前数据已经显示，RBSOG 在压力方差与结构稳定性方面表现出一定优势，因此问题的关键不在于当前原型是否已经“跑得最快”，而在于其是否表现出值得进一步工程优化的潜力。从解决思路上看，下一阶段应继续加强内核级优化，包括进一步减少 Python 层循环、优化长程采样数据流、增加更深层的 JIT 或向量化支持，并视条件逐步考虑将关键数值核心迁移至更高性能后端，以便更加真实地评估该方法的吞吐能力。

第三，当前统计样本量和轨迹长度仍偏有限，尚不足以支撑更高强度的统计结论。当前实验多数采用较短步长范围和较少随机种子，其优点在于便于快速迭代和验证流程，但缺点在于结果容易受到样本波动影响，特别是在随机批量方法中，批次采样和随机初值本身就会带来额外离散性。若希望进一步形成具有说服力的论文级结果，就必须显著增强统计样本量和轨迹长度。对此，后续应将实验设计分层推进：一方面，通过增加种子数来提高统计稳健性；另一方面，通过延长轨迹长度来增强稳态统计量的可信度。在资源受限的情况下，也可采用“短轨迹多重复 + 长轨迹关键验证”相结合的方式，以平衡计算成本与统计质量。

第四，当前对于 RBSOG 复杂度优势的论证仍以趋势性观察为主，尚缺少足够系统的规模扫描证据。课题的长期目标在于面向超大规模体系中的长程静电求解，因此仅凭当前小体系下的性能对比，仍难以充分说明其在大规模情形中的扩展潜力。特别是与 FFT 相关的通信瓶颈问题，在单机小规模原型中无法完全显现。因此，现阶段还需要避免对“大规模性能优势”作过度外推。相应地，解决思路是增加体系规模扫描实验，在不同粒子数条件下比较 PPPM 与 RBSOG 的平均步耗时和关键统计量，并通过对数坐标斜率拟合给出经验复杂度证据。只有在补齐这部分实验后，课题才能在“通信规避潜力”和“规模扩展趋势”方面提出更扎实的论断。

第五，当前精度验证工作仍以统计量与结构代理指标为主，对参考解误差的系统分析尚不充分。虽然已有 direct solver 可作为小规模参考，但目前尚未围绕力误差、势能误差以及压力偏差等指标开展系统参数扫描研究。因此，对于 RBSOG 在不同参数条件下的误差来源，目前仍缺少足够清晰的定量拆分。例如，当前尚需进一步回答如下问题：误差主要来自随机批量采样，还是来自高斯和近似本身；在较小批量下为何结构稳定性仍可维持；不同 SOG 项数与截断半径对误差与成本的影响边界如何。这些问题的解决，需要后续专门设计小体系参考实验，将“速度—方差”分析进一步拓展为“速度—方差—精度”联合分析。

第六，当前学术报告已经具备较好的技术基础，但与正式会议投稿稿件相比仍有进一步提升空间。现有文稿已经能够较完整地呈现研究背景、方法设计、实验结果和可复现性流程，但在论证严谨性、相关工作覆盖度、数学推导规范性和附录信息完整性等方面，仍需进一步打磨。尤其是在未来考虑正式投稿时，还需要根据目标会议模板和评审偏好补充更明确的贡献陈述、更严格的消融分析、更清晰的误差讨论以及更系统的 artifact checklist。对此，后续工作的重点不应是重写已有文稿，而应基于当前自动化写作与资产生成框架持续迭代，使报告逐步升级为真正具备投稿质量的论文草稿。

总体而言，当前存在的问题更多体现为课题由“方法原型验证”向“强证据研究结论”过渡过程中的自然挑战，并不影响既有研究路线的合理性。相反，这些问题的清晰暴露恰恰说明课题前期目标已经基本实现，研究已顺利进入更高层次的深化阶段。后续工作的关键，不在于重新确立方向，而在于通过更加系统的实验、更严谨的统计和更强的工程化实现，将当前已建立的原型优势转化为更具学术说服力的成果。

## 三、下一步的工作计划和研究内容

结合当前研究进展和问题分析，下一阶段工作将围绕“增强实验真实性、提升统计可靠性、补充复杂度证据、完善精度分析、推进论文成稿”五条主线展开。总体思路是，在保持现有原型框架和评测体系稳定的基础上，逐步提升研究工作的深度、广度与证据强度，使课题从“原型可行”进一步迈向“结论可靠、结果可发表、方法可扩展”的发展阶段。

首先，下一阶段将优先加强实验设计的统计稳健性。当前已有的 benchmark 和 sweep 管线已经具备多随机种子、多参数扫描和结果自动汇总能力，因此接下来的重点是扩大实验样本规模。具体而言，将适当增加随机种子数，并在关键参数组合下延长模拟步数，以便更加可靠地估计平均每步时间、压力方差、结构稳定性指标和相关统计波动。同时，还将进一步引入更规范的置信区间、效应量以及必要的显著性分析，从而使 RBSOG 与 PPPM 的差异不仅体现在趋势层面，而且能够形成更加严格、清晰的统计证据。这部分工作将直接提升课题结果的可信度，也是后续撰写正式论文时不可或缺的基础支撑。

其次，将系统开展规模扫描和复杂度分析实验。考虑到课题的核心科学问题之一在于为更大规模体系提供新的长程静电求解思路，仅在单一规模下比较 PPPM 与 RBSOG 是远远不够的。下一步计划在现有框架下构建不同粒子数规模的系列实验，比较两类方法在不同体系规模下的步耗时、方差和稳定性变化，并通过拟合分析得到经验复杂度趋势。与此同时，还将把批量大小 P 的扫描与体系规模 N 的变化结合起来，研究推荐 P 是否随系统规模呈现稳定规律，进而为将来构建自适应参数选择策略提供依据。这项工作将在理论意义和应用意义上同时增强课题的深度。

第三，将进一步补充精度验证与误差分解研究。当前阶段的结果已经说明 RBSOG 在若干统计量上具有可研究价值，但要形成更完整的算法评价体系，还必须系统分析其相对于参考解的误差行为。下一步将以 direct solver 为小体系参考，围绕批量大小、SOG 项数、截断半径等关键参数开展误差扫描实验，重点分析力误差、势能误差、压力偏差以及结构指标偏差的变化规律。同时，将尝试区分误差来源中“高斯和核近似误差”和“随机批量采样误差”的贡献比例，以便更加准确地理解 RBSOG 方法的数值特性。通过这一系列工作，现有研究将从“性能与方差比较”为主，逐步扩展为“性能—方差—精度统一评价”的更完整框架。

第四，将推进实验对象向更真实体系过渡。当前代理体系在方法开发早期发挥了重要作用，但为了增强研究结论的生物物理相关性，后续应逐步引入更贴近真实模拟场景的膜体系。具体计划是先尝试使用更标准化的膜构型作为输入，在尽量保持现有分析流程不变的前提下，对面积/脂质、膜厚、压力涨落等关键观测量进行验证。待这一阶段工作稳定后，再进一步考虑更复杂的组分、边界条件或体系环境，以评估 RBSOG 在更加复杂电荷分布和结构约束下的稳定性与适用性。该方向的推进将使课题逐渐由“算法原型研究”过渡到“面向真实生物体系的算法验证研究”。

第五，将继续加强数值实现与工程优化。虽然当前原型已完成一轮近邻表、向量化和 JIT 优化，但 RBSOG 的性能潜力仍有较大挖掘空间。下一步将在现有代码基础上进一步优化长程采样流程，减少中间数据复制和解释器级调度开销，提升批处理效率与内存访问友好性。同时，也将视实际进展评估是否采用更适合科学计算原型优化的后端工具，以便在不牺牲研究灵活性的前提下提高数值效率。从长远看，还应逐步考虑与主流分子动力学平台的接口适配问题，为未来将当前方法迁移到更高性能、更接近生产环境的模拟软件中做好准备。

最后，在学术成果凝练与论文写作方面，下一阶段将以现有 LaTeX 报告为基础，继续朝正式会议投稿稿件的标准推进。一方面，将补充更完整的相关工作综述、更清晰的方法贡献总结以及更系统的实验对照与消融分析；另一方面，也将同步完善图表规范、附录材料、实验命令、随机种子记录和 artifact checklist，提升文稿的完整性和可复现性。待实验结果积累到一定程度后，可围绕“基于高斯和分解与随机批量采样的长程静电分子动力学原型算法”形成一篇方法学论文，作为当前阶段最主要的学术输出目标。

总体来看，下一步工作计划既强调继续做深算法与实验本身，也强调同步提升结果表达和学术组织质量。其核心目标是把当前已经建立起来的原型系统和阶段性结果，进一步发展为具有明确论证链条、较高复现性和较强学术竞争力的研究成果。只要后续各项工作按计划稳步推进，本课题有望在方法创新、实验体系建设和论文产出三个层面同步取得更加显著的进展。
